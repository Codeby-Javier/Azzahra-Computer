SETUP DAN RUN AZZAHRA WEB
========================

REQUIREMENT:
============
1. PHP 7.4+ dengan MySQLi extension
2. MySQL Server running
3. Database: azzahra (sudah ada di mou_database.sql)

SETUP PERTAMA KALI:
===================
1. Import database:
   - Buka phpMyAdmin (http://localhost/phpmyadmin)
   - Import file: mou_database.sql
   - Database akan dibuat: azzahra

RUNNING THE WEB:
================
Cara 1 - Menggunakan easy_run.bat (RECOMMENDED):
  1. Double-click file: easy_run.bat
  2. Tekan ENTER
  3. Browser otomatis membuka: http://localhost:8000
  4. Untuk stop: Tekan Ctrl+C di command prompt

Cara 2 - Manual dengan command:
  1. Buka Command Prompt di folder project
  2. Jalankan: php -S localhost:8000 router.php
  3. Buka browser: http://localhost:8000
  4. Untuk stop: Tekan Ctrl+C

AKSES WEB:
==========
- URL: http://localhost:8000
- Default Page: Login (Auth Controller)

TROUBLESHOOTING:
================

Problem: "Cannot connect to database"
Solution:
  1. Pastikan MySQL Server running
  2. Cek database name: azzahra
  3. Cek user: root (atau sesuaikan di config/database.php)
  4. Jalankan: mou_database.sql di phpMyAdmin

Problem: "Page tidak ditemukan / 404"
Solution:
  1. Pastikan router.php berada di root folder
  2. Restart PHP server
  3. Cek console browser (F12) untuk error message

Problem: "CSS/JS tidak dimuat"
Solution:
  1. Ini normal - assets statis masih dimuat
  2. Refresh page (F5)
  3. Clear browser cache (Ctrl+Shift+Delete)

SUPPORT:
========
Jika ada masalah, check:
1. application/logs/ - untuk error log
2. Browser console (F12) - untuk JavaScript error
3. Pastikan port 8000 tidak sedang digunakan
